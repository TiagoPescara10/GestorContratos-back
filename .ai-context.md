# AI Development Context - Gestor de Contratos

## Project Overview
This is a contract management system built with React, Vite, and Chakra UI. The application manages rental contracts with features for creating, viewing, editing, and tracking contracts.

## Tech Stack
- **Frontend**: React 19.2.4 with Vite 8.0.1
- **UI Framework**: Chakra UI 2.10.9 with Emotion
- **Routing**: React Router DOM 7.13.2
- **Icons**: React Icons 5.6.0
- **Animations**: Framer Motion 12.38.0

## Project Structure
```
src/
|-- api/           # API client and endpoints
|-- components/    # Reusable React components
|-- pages/         # Page-level components
|-- routes/        # Routing configuration
|-- utils/         # Utility functions
|-- DocumentosTXT/ # Text documentation
```

## Key Features
- Contract CRUD operations
- Dashboard with statistics
- Contract search and filtering
- Protected routes with authentication
- Contract calculations (IVA, increases, etc.)
- PDF generation capabilities

## API Integration
- Uses a REST API with client in `src/api/client.js`
- Contract endpoints in `src/api/contratos.js`
- Index calculations in `src/api/indices.js`

## Component Architecture
- **Sidebar**: Main navigation component
- **CardContrato**: Contract card display
- **FormContrato**: Contract creation/editing form
- **ContractStatCard**: Statistics display
- **ContratoCompleto**: Full contract details

## Important Utilities
- `formatearMonto`: Currency formatting
- `capitalizar`: Text capitalization
- `generarMeses`: Month generation logic
- `aplicarAumento`: Contract increase calculations

## Development Guidelines
- Use Chakra UI components consistently
- Follow React hooks best practices
- Maintain responsive design principles
- Implement proper error handling
- Use semantic HTML structure
